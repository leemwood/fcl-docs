# FCL_CE 教程文档

fcl玩家社区的官方文档网站，专注于 Minecraft Java Edition for Android。

## 简介

本项目是 fcl玩家社区 的教程文档站点，帮助用户了解和使用 FCL 启动器及其相关插件、Vulkan 模组等内容。

## 技术栈

- [Docusaurus 3](https://docusaurus.io/) - 现代化静态网站生成器
- React 19
- TypeScript

## 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm start

# 构建生产版本
npm build

# 预览构建结果
npm serve
```

## 文档结构

```
docs/
├── intro.md           # 入门介绍
├── fcl/               # FCL 启动器相关文档
├── plugins/           # 插件文档
└── vulkan/            # Vulkan 模组文档
```
