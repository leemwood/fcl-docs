这是一份基于视频内容的 **Zalith Launcher 2 (ZL2)** 详细保姆级教程。ZL2 是一款专为 Android 设备打造的 Minecraft Java 版启动器，支持模组、光影和整合包。

### 第一阶段：下载与安装

1.  **选择版本：**
    *   下载时你会看到不同后缀的 APK 文件，请根据你的手机处理器选择：
        *   **`arm64-v8a`**：**最常用**。适用于近几年购买的绝大多数安卓手机（64位系统）。
        *   **`armeabi-v7a`**：适用于 2016 年前的旧机型或 32 位系统手机。
        *   **`x86` / `x86_64`**：通常用于安卓模拟器或特殊机型。
    *   *提示：如果不确定，可以下载体积最大的通用版本（Universal），但文件较大。*

2.  **安装启动器：**
    *   点击安装 APK。
    *   首次打开会提示安装必要组件（authlib-injector, Java环境等），点击**同意并安装**。
    *   等待解压完成，即可进入主界面。


### 第二阶段：登录账号

点击主界面右侧的**添加账号**，有三种登录方式：

1.  **微软账号（正版）：**
    *   前提：你已在微软商店购买 Minecraft Java Edition，并创建了游戏 ID（Xbox档案）。
    *   操作：点击“微软账号” -> 点击“登录” -> 复制弹出的验证码 -> 在浏览器中输入验证码 -> 登录微软账号 -> 授权 -> 返回启动器等待验证完成。

2.  **离线账号（非正版/盗版）：**
    *   操作：点击“离线账号” -> 输入游戏 ID。
    *   *注意：ID 仅支持大小写英文字母、数字和下划线。包含其他字符可能导致高版本无法联机。*

3.  **外置登录（皮肤站账号）：**
    *   操作：点击左下角**添加认证服务器** -> 输入服务器 API 地址（例如 LittleSkin: `https://littleskin.cn/api/yggdrasil`） -> 确认。
    *   返回后，点击刚才添加的服务器下方的按钮 -> 输入你在皮肤站注册的邮箱/用户名和密码 -> 登录。

### 第三阶段：启动器主要设置

点击右上角**齿轮图标**进入设置：

1.  **渲染器设置（关键）：**
    *   **Krypton Wrapper**（默认）：兼容性好，支持大部分模组和 Iris 光影。
    *   **GL4ES**：仅推荐低版本 MC 使用，运行模组需手动改配置。
    *   **Mobile Glues**（推荐高版本/光影/整合包）：性能最强。
        *   *如何安装 Mobile Glues：*
            1.  下载并安装 `MobileGlues` 的 APK 应用。
            2.  打开 Mobile Glues APP，点击“修改渲染器设置”。
            3.  点击“确认”进入授权界面 -> 点击右上角新建文件夹 -> 命名为 **`MG`** -> 进入 MG 文件夹 -> 点击“使用此文件夹”并允许访问。
            4.  回到 ZL2 设置中，即可选择 Mobile Glues。

2.  **游戏窗口分辨率：**
    *   默认 100%（即屏幕原生分辨率）。
    :::tip 建议
    如果手机屏幕分辨率很高（如 2K 屏），建议调至 **50%** 左右，能显著降低渲染压力，提升帧率。
    :::

3.  **游戏设置：**
    *   **全局版本隔离**：默认开启。**强烈建议保持开启**，这样每个版本的模组、存档、配置互不干扰，防止游戏崩溃。
    *   **Java 环境**：默认“自动选择”。除非玩特殊整合包（如 1.12.2 Cleanroom 需 Java 21），否则无需修改。
    *   **内存分配**：默认 2048MB。可根据手机剩余内存适当增加，但**不要分配过高**（如超过手机总内存的一半），否则会导致系统卡顿或闪退。

4.  **启动器设置：**
    *   **下载源**：默认“优先使用官方源”。如果在国内下载慢或失败，可切换为“优先使用镜像源”。

### 第四阶段：按键布局

*   在设置左侧滑动找到**控制管理**。
*   你可以点击**导入”**加载别人分享的布局，或者点击**创建”**自定义按键位置、大小和功能。


### 第五阶段：安装游戏本体

1.  点击主界面右上角**下载图标**。
2.  点击左侧**游戏**。
3.  选择版本：
    *   推荐尝试模组生态丰富的版本，如 **1.20.1** 或经典的 **1.12.2**。
    *   原版玩家任意选择即可。
4.  选择模组加载器（点击版本号右侧的下拉箭头）：
    *   **OptiFine**：老牌优化模组，自带光影，但兼容性较差。推荐低版本或纯原版使用。
    *   **Forge**：老牌模组加载器，模组数量最多。
    *   **Fabric**：轻量级，性能优化模组（如 Sodium）多基于此。**注意：使用 Fabric 必须同时勾选下方的 `Fabric API`。**
    *   **Quilt**：基于 Fabric 的分支，兼容大部分 Fabric 模组。
    *   *总结：玩大型模组选 Forge/NeoForge；追求极致性能选 Fabric。*
5.  点击下载，等待安装完成。


### 第六阶段：安装模组

**方法一：ZL2 内部下载（推荐）**
1.  确保已安装对应版本的 Forge 或 Fabric。
2.  点击右上角下载图标 -> 左侧点击**模组**。
3.  在右侧筛选对应的**游戏版本**和**模组加载器**（如 1.20.1 + Forge）。
4.  搜索模组（如“传送石碑”），点击进入详情页。
5.  选择对应版本点击下载。
    :::danger 注意
    如果模组有**前置依赖**（如 Balm），弹窗会提示，务必勾选一起下载，否则游戏会崩溃。
    :::

**方法二：外部手动导入**
1.  在 MC百科或其他网站下载模组的 `.jar` 文件。
2.  打开文件管理器（如 MT管理器）。
3.  找到下载的文件，复制。
4.  粘贴到 ZL2 的游戏目录：
    `Android/data/com.movtery.zalithlauncher.v2/files/minecraft/versions/版本名/mods`
    *(如果没有 mods 文件夹，请新建一个)*



### 第七阶段：加载光影

**方案 A：Iris + Sodium (推荐高版本)**
*   **Fabric版：** 安装 `Iris` 和 `Sodium` 模组。
*   **Forge版：** 安装 `Embeddium` (Sodium的移植版) 和 `Oculus` (Iris的移植版)。

**方案 B：OptiFine**
*   在安装游戏版本时直接选择 OptiFine。

**启用光影步骤：**
1.  在 ZL2 下载中心 -> 左侧点击**光影包** -> 下载光影（如 BSL）。
2.  启动游戏 -> 选项 -> 视频设置 -> 光影包。
3.  选择刚才下载的光影并应用。
    *   *外部导入：* 将下载的 `.zip` 光影文件放入 `shaderpacks` 文件夹（路径同 mods 文件夹，只是把 mods 换成 shaderpacks）。
        :::tip 提示
        如果卡顿，请在 ZL2 设置里降低“游戏窗口分辨率”。
        :::



### 第八阶段：安装整合包

**方法一：ZL2 内置源下载**
1.  下载中心 -> 左侧点击**整合包**。
2.  找到喜欢的整合包（如 Better MC），选择版本安装。
:::warning 警告
大多数整合包是为电脑设计的，手机端运行可能会崩溃或卡顿。建议寻找专门针对手机端优化过的移植版整合包。
:::

**方法二：外部导入（推荐）**
1.  下载整合包文件，格式通常为 **`.mrpack`** 或 **`.zip`**。
    *   *注意：* ZL2 **不支持**导入“懒人包”（即解压后直接包含 .minecraft 文件夹的压缩包）。
2.  在 ZL2 下载中心 -> 整合包页面 -> 点击右上角**“导入整合包”**。
3.  选择你下载的文件 -> 确认安装。启动器会自动下载所需的模组和资源。


### 第九阶段：远程联机（内置 EasyTier）

ZL2 内置了联机功能，无需第三方软件，离线账号也能联。

**房主操作：**
1.  在 ZL2 主界面右上角点击**多人联机图标** -> 开启“启用多人联机”。
2.  启动游戏，进入你想联机的存档。
3.  按 Esc -> **对局域网开放** -> 创建局域网世界。
4.  打开 ZL2 悬浮窗 -> 点击**联机菜单** -> 选择**“我想当房主”**。
5.  启动器会自动扫描并创建房间，复制生成的**邀请码**发给朋友。

**房客（朋友）操作：**
1.  同样在 ZL2 开启“启用多人联机”。
2.  启动游戏（版本和模组必须与房主完全一致），**停留在主界面**，不要进存档。
3.  打开 ZL2 悬浮窗 -> 点击**联机菜单** -> 选择**“我想当房客”**。
4.  粘贴房主发来的**邀请码** -> 确认。
5.  显示“已加入房间”后，返回游戏主界面 -> 点击**多人游戏**。
6.  在列表中看到房主的房间（通常显示为局域网世界），双击加入即可。
