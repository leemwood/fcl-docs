# 公益玩家社社群群规

    欢迎加入由公益玩家社搭建的第三方社区交流群，本群为了便利玩家交流而建立，本群隶属于公益玩家社！

群规

1. 刷屏类：本群禁止任何刷屏行为！
  （无意义文字15行 重复消息超过或等于5条 多人合作刷屏 图片表情包刷屏）

2. 精神文明类：本群禁止出现违背精神文明类的聊天内容出现！
 （辱骂他人 打架约架 说谎造假 造谣传谣）

3. 法律法规类：本群严禁交流触犯法律法规的消息！
 （色情低俗 开户开盒 贩毒吸毒 赌博 未经他人允许传播他人隐私等）

4. 政治历史类：本群禁止交流任何国家任何时期的政治历史类消息
 （不包括具有正能量精神的政治历史类消息）

5. 宣传类：本群禁止宣传类消息！
 （广告 营销 游戏包 虚拟单 链接 机器人）

6. 邀请类：本群禁止任何邀请类消息！
 （其他群聊及频道：群号或频道号 二维码 邀请链接）

7. 游戏消极类：禁止发消极任何游戏的消息！
 （科技外挂 破解版游戏 游戏账号交易 游戏破解软件）

8. 游戏讨论类：本群可以适当讨论其他游戏内容，但不得影响其他玩家正常交流！

## 处罚规则

违反1.5.6.8.群规的成员将禁言处罚（禁言时间：15分钟-3天）

违反2.7.群规的成员将禁言或踢出群聊处罚（禁言时间：3天-10天）

违反3.4.群规的成员将禁言或踢出群聊（禁言时间7天-踢出群聊）

处罚：翻倍处罚法，处罚力度为前一次二倍，第一次按最小算，不得超过最大期限！

解释权归群主和管理所有，望群员们遵守群规！

注意：本群管理员踢出群员需要得到群主的批准，不得擅自做主！

## 管理员行为规范

1. 管理员不得带头违反群规
2. 管理员不得滥用权利
3. 管理员行使权利时不得超过权利的范围
4. 管理员不能看见违规而不做处理
5. 管理员不得背叛群主
6. 管理员不得擅自修改群昵称及群头像
7. 管理员踢人前必须提前在管理群报备
8. 管理员之间不得发生内斗
9. 管理员必须严格按照群规管理群聊及频道
10. 管理员必须听从群主安排
11. 群主有权收回管理员权限
