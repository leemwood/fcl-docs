# 下载

## 下载启动器

 - [fcl下载站](https://foldcraftlauncher.cn/down.html)  
 提供fcl，zl，zl2，各种渲染器，JAVA和驱动。
 - [zl官网](https://www.zalithlauncher.cn/download)  
 只能下载zl和zl2
 - [柠枺镜像](https://mirror.lemwood.icu)  
 提供fcl，zl，zl2，shizuku，mg渲染器和hmcl（电脑版启动器）


## 下载工具

- [MT管理器](https://mt2.cn)  
mt管理器不只是文件管理器，它的功能太强大了。
- [shizuku](https://mirror.lemwood.icu/download/shizuku/v13.6.0/shizuku-v13.6.0.r1086.2650830c-release.apk)  
仅次于root的安卓adb工具，配合mt管理器食用更佳😋[官网地址](https://shizuku.rikka.app/zh-hans/)
- [[DH]遥远的地平线手机专用补丁版](https://dh.lemwood.cn)  
自动注入 libzstd-jni 手机版补丁，支持 aarch64 架构，为手机端 Minecraft 提供完美支持，还有自带翻译支持。
